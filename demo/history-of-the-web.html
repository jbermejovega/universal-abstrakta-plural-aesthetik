A Brief History
of the World Wide Web

From a scientist's proposal at CERN to the global platform that connects billions — the story of how the web was born, evolved, and changed everything.

1989–Today
  1. 89
    March 1989

    Tim Berners-Lee writes the proposal

    While working at CERN, Sir Tim Berners-Lee submitted “Information Management: A Proposal”, outlining a hypertext-based system to help scientists share information across networks. His boss scribbled “Vague, but exciting” on the cover — and the web was conceived.

  2. 91
    August 1991

    The first website goes live

    The world's first website — info.cern.ch — explains what the World Wide Web is, how to create web pages, and how to browse them. It runs on a NeXT computer at CERN, and the source code is released into the public domain in 1993.

  3. 93
    1993–1994

    Mosaic, Netscape, and the W3C

    The Mosaic browser brings images and text together in the same window for the first time, sparking mainstream adoption. Netscape Navigator follows in 1994, and Berners-Lee founds the World Wide Web Consortium (W3C) at MIT to steward open web standards.

  4. 95
    1995–1996

    JavaScript and CSS are born

    Brendan Eich creates JavaScript in just 10 days at Netscape, giving developers a way to script the browser. A year later, Håkon Wium Lie publishes CSS Level 1, separating presentation from structure and laying the foundation for modern web design.

  5. 98
    September 1998

    Google is founded

    Larry Page and Sergey Brin launch Google from a garage in Menlo Park. Their PageRank algorithm revolutionizes search, and Google quickly becomes the web's dominant gateway — shaping how billions of people discover information online.

  6. 04
    2004–2005

    Web 2.0 and AJAX

    Jesse James Garrett coins the term AJAX, enabling pages to update without full reloads. Firefox challenges Internet Explorer's dominance, and platforms like Flickr, YouTube, and Facebook usher in the era of user-generated content and social media.

  7. 07
    June 2007

    The iPhone launches the mobile web

    Apple's iPhone introduces a full desktop-class browser on a phone. Responsive design becomes essential as developers grapple with touch interfaces, viewport sizes, and mobile-first experiences. The web is no longer something you sit at a desk to use.

  8. 08
    2008–2014

    Chrome and HTML5

    Google releases Chrome with a blazing-fast V8 JavaScript engine in 2008. Work on HTML5 accelerates, bringing native audio, video, canvas, and semantic elements. The HTML5 Recommendation is published in 2014, cementing the modern web platform.

  9. 15
    2015

    ES6 and the JavaScript renaissance

    ECMAScript 2015 (ES6) is the largest update to JavaScript since its creation — adding arrow functions, classes, modules, promises, and template literals. Paired with frameworks like React, Angular, and Vue, the web becomes a full-fledged application platform.

  10. 20
    2020–Present

    The modern web platform

    WebAssembly, CSS Grid, Container Queries, and the View Transitions API push what's possible in the browser. Accessibility standards like WCAG 2.2 ensure the web works for everyone. AI-assisted development, edge computing, and real-time collaboration define the current era — and the web's story is still being written.
