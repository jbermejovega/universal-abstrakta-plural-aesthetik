## Git normalization

Merge, squash, and rebase are context-typed renormalization operations:

- **merge** preserves plural branch ancestry;
- **squash** projects a reviewed branch to one certified persistence event;
- **rebase** changes the presentation base while preserving the patch semantics;
- none is valid when the resulting trace loses provenance, witnesses, or branch-boundary meaning.
